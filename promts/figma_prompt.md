# Prompt for Figma Make AI --- Image Editor Web App

**Project title:** Simple Image Editor --- web app (upload → preview →
basic edits → download)

**One-line summary:** Design a clean, user-friendly responsive web app
that lets users upload images from their computer, preview them, apply a
set of basic edits (blur, grayscale, brightness, contrast, flip,
crop/rotate), and download the result as JPG or PNG.

------------------------------------------------------------------------

## Goals & users

-   **Primary goal:** Fast, easy image edits for non-technical users.
-   **Target devices:** Desktop (primary), tablet, mobile (responsive).
-   **Tone & style:** Modern, minimal, friendly, usability-focused.

------------------------------------------------------------------------

## Required screens / artboards

1.  **Landing / Upload screen**
2.  **Editor screen**
3.  **Crop / Rotate modal**
4.  **Export / Download modal**
5.  **Empty/error states**
6.  **Mobile responsive variations**

------------------------------------------------------------------------

## Key features

-   Upload (drag & drop or file picker)
-   Image preview
-   Editing tools:
    -   Blur (slider)
    -   Grayscale (toggle)
    -   Brightness (slider)
    -   Contrast (slider)
    -   Flip (horizontal/vertical)
    -   Crop or Rotate
-   Download as JPG or PNG
-   Undo / Redo / Reset
-   Live preview updates
-   Vietnamese labels by default

------------------------------------------------------------------------

## Visual system & components

-   Modern neutral palette + accent
-   Typography tokens (Inter / Noto Sans)
-   Components: buttons, modals, sliders, toggles, icons
-   Responsive layout for desktop/tablet/mobile

------------------------------------------------------------------------

## Accessibility

-   WCAG AA contrast
-   Keyboard navigation & ARIA labels
-   Editable alt text

------------------------------------------------------------------------

## Developer handoff expectations

------------------------------------------------------------------------

## Design style

-   Minimal UI, soft shadows, rounded corners
-   Smooth transitions
-   Optional light/dark theme tokens

------------------------------------------------------------------------

## Acceptance criteria

-   Fully clickable upload → edit → export prototype
-   Reusable components & tokens
-   Vietnamese UI labels with English alternatives
